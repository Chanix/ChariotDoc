应用类型 ​

全部后端，远程HTTPD ​

此种模式下，和浏览器没什么区别，所有的代码都在后端。 同时也可以通过JSAPI提供扩展的功能，可以看成是一个扩展后的网页浏览器。 这是最传统的模式。

前后端分离，本地HTTPD ​

前后端分离，Chariot会启动一个本地httpd服务器（端口随机）供前端使用。前端通过API接口与后端通信。 这是目前主流的web应用模式。
