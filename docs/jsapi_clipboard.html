Chariot.clipboard.is_available() ​

系统剪贴板是否可操作

返回值说明

javascript
await __C.clipboard.is_available();

Chariot.clipboard.get() ​

获取系统剪贴板中的值（仅支持字符串）

返回值说明
string剪贴板中存储的值

Chariot.clipboard.set(text) ​

参数类型说明
