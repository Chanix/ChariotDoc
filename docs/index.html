什么是 Chariot?
快速开始
JS API

基于 Web 技术

开发人员可以使用熟悉的 Web 前端技术来创建用户界面和业务逻辑。复用现有资源，快速上手，学习曲线平缓。

跨平台支持

支持 Windows、MacOS 和 Linux 等不同操作系统，开发人员可以使用同一套代码库构建跨平台的桌面应用程序。

内置扩展

内置扩展对象，JavaScript 可直接调用，包括原生操作系统模块的访问、本地资源的操作等，以支持开发各种类型的桌面应用程序。

易于打包和分发

简单方便的打包和分发机制，分发对应平台的 Chariot 可执行文件和项目包即可。更可以打包成可执行文件推广到用户手中。
